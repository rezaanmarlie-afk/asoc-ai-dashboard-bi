# Version 5.4 Preview - Benefits Management Suite

This build removes the AI capability and focuses on making Benefits Intelligence more enterprise-grade.

## New / improved

- Professional Benefits Intelligence page
- Portfolio filters by business unit, domain, benefit and owner
- Executive value cards
- Benefit 360 sections with current, target, risk and opportunity
- Click any chart to open a large executive review modal
- BI Admin upgraded to Benefits Management Suite
- Admin tabs:
  - Benefit Register
  - Benefit History
  - Contributors
  - Evidence
  - Approvals
  - Import / Export
- Enriched editable fields:
  - benefit ID
  - category
  - strategic theme
  - business unit
  - domain
  - AI use case ID and name
  - sponsor
  - owner
  - financial owner
  - status
  - stage
  - maturity
  - confidence
  - source/evidence
  - systems/models
  - risks/opportunities/dependencies
  - approval status
- CSV import/export now supports enriched BI data
