# Version 5.4.5 - Status Exclusion and Heatmap Hours Saved

Added:
- AI Portfolio Heatmap now includes an **Hours saved** column.
- Dashboard Status View now supports excluding one or more statuses from calculations.
- Excluded statuses are removed from:
  - KPI cards
  - Executive meter gauges
  - Portfolio charts
  - AI Portfolio Heatmap
  - `/api/summary`

Usage:
1. Open the dashboard.
2. Use **Show status** to view all statuses or one selected status.
3. Use **Exclude from calculations** to remove statuses such as `AI Opportunity`, `Planned`, `Sandbox`, or `Retired` from totals.
4. Use Ctrl+click to select multiple exclusions.
