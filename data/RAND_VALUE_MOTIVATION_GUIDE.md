# Rand Value Motivation

The Rand values are mock / directional estimates for dashboard testing until ASOC actuals are available.

Recommended formula:

Annual Benefit =
((Baseline minutes - AI-assisted minutes) × Monthly volume × 12 ÷ 60 × Fully-loaded hourly rate)
+ Operational impact proxy
+ Cost avoidance
+ Reuse benefit

Portfolio ROI =
((Annual Benefit - Implementation Cost) / Implementation Cost) × 100

Replace mock values with actual baseline, current and target KPI measurements during benefits tracking.
